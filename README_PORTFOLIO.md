# Inventory Demo (Portfolio)

This project is a portfolio-friendly clone of an internal inventory workflow.

## Privacy notes

- Branding is replaced with neutral demo identity.
- The app is configured with a demo base path: `/inventory-demo`.
- Use only synthetic data and demo credentials.

## Quick start

1. Install dependencies:
   - `npm install`
2. Configure environment:
   - Copy `.env.example` to `.env`
3. Run development server:
   - `npm run dev`
